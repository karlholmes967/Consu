# Consu

A personal media tracking app for movies, TV, games, books, and music.

Currently in closed development. Public release planned via Apple App Store and Google Play.

**Tech stack:** React 18, Vite, Firebase, Cloudflare Workers.

**Data attribution:** Uses TMDB, RAWG, Google Books, NYT, OMDb, Last.fm, MusicBrainz, Deezer, iTunes, Open Library, and Steam. Full credits available in the app's Settings → Attribution section.

Source is not currently public.
